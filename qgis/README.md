# QGIS Project

This folder contains QGIS project files used for spatial analysis and map creation.

## Contents

- **Sharmin_Project_urban.qgz**  
  Main QGIS project file containing:
  - District boundaries
  - Playground locations
  - Processed attribute tables
  - Choropleth map styling

## Key QGIS Operations

- Attribute table cleaning and field selection
- Calculation of playgrounds per 1,000 children
- Choropleth mapping using graduated symbology
- Map layout design and export to PDF
